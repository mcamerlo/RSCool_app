Valores em falta

  • Detectar NAs numa variável: 
table (is.na(x))


  • Tabela que não exclua NAs:
table (x, exclude = FALSE)


  • Calcular média duma variável com NAs: 
mean(x, na.rm = TRUE)


  • Eliminar observações com NA numa variável: 
Df <-subset( Df   ,   !is.na (sexo))


  • Eliminar obs com NA em todas as variables da base Df: 
Df <- na.omit (Df)


  • Recodificar NAs em 0: 
y[is.na(y)] <- 0              # recodifica “NA” de “y” em “0”


  • Recodificar como NA: 
(D$x[D$x==88] <- NA           # recodifica x==88  em NA


  • Listar linhas com NAs: 
Df[!complete.cases(Df),]


  • Criar nova base sem NAs: 
DF_new <- na.omit(DF)

FACTORS

  • Factorizar uma variável numérica: 
cut (x, breaks=3)


  • Factorizar uma variável não factor: 
v <- c(0, 1, 1, 0, 5,5, 0)

f <- factor (v)


  • Visualizar ‘levels’ de uma factor:
levels(f) 
[1] "0" "1" "5"


  • Atribuir ‘labels’ a uma factor:
f <- factor(f,
            levels = c(0,1, 5),
             labels = c("azul", "verde", "cinzento"))


  • Recodificar em 2 categorias:
f.2 <- ifelse(f=="azul",  "A", "nonA")


  • Recodificar em 3 categorias:
age <- c(9, 9, 11, 14, 16, 30, 50)

age.3 <- NA
age.3  [age < 10]             <- "child"
age.3  [age >= 10 & age < 20] <- "teenager"
age.3  [age >= 20]            <- "adult"

f <- relevel (f, "verde")   # ver ordem de "Levels"

  • Agregar uma categoria: 
f <- factor(f, levels = c(levels(f), "BRANCO"))
f

  • Invertir a ordem dos ‘levels’:
f_rev  <- factor(f, levels = rev(levels(f)))

  • Reordenar ‘levels’
f <- factor(f, levels = c("verde", "AZUL"))

  • Ordenar levels de uma factor baseado na ordem das linhas numa base
    mais info
library(dplyr)

Df <- mutate(Df, col = factor(col, col))


library(dplyr)

Df <- data.frame(
  num = 5:1,
  word = factor (c("five", "four", "three", "two", "one")))
levels(Df$word) # levels are alphabetical by default
[1] "five"  "four"  "one"   "three" "two"  
Df$word # right now the order is 5,4,3,2,1
[1] five  four  three two   one  
Levels: five four one three two
Df <- Df %>%
  arrange(num) %>%   # rearrange the Df in the order we want (1,2,3,4,5)
  mutate(word = factor(word, word)) # this line reorders the factor in the same order

levels(Df$word) 
[1] "one"   "two"   "three" "four"  "five"

STRINGS

strings <- c("maçã", "banana", "laranja", "uva", "kiwi")


Acessar elementos individualmente

strings[1]
[1] "maçã"


Acessar vários elementos

strings[c(2, 4)]
[1] "banana" "uva"


Combinação de Strings:

paste("Eu gosto de", strings[1])
[1] "Eu gosto de maçã"

Alterar Maiúsculas/Minúsculas:

toupper(strings)
[1] "MAÇÃ"    "BANANA"  "LARANJA" "UVA"     "KIWI"   
tolower(strings)
[1] "maçã"    "banana"  "laranja" "uva"     "kiwi"

Extração de Substrings:

substr(strings[1], start = 1, stop = 3)
[1] "maç"

Length da String:

nchar(strings[1])
[1] 4


Verificar se uma string contém um padrão específico

contem_maca <- grepl("maçã", strings)
print(contem_maca)
[1]  TRUE FALSE FALSE FALSE FALSE


Filtrar strings com base em uma condição

frutas_longas <- strings[nchar(strings) > 5]
print(frutas_longas)
[1] "banana"  "laranja"
