Statistical Tests


Testes-t

Sintaxe por defeto

t.test(x, y = NULL,
       alternative = c("two.sided", "less", "greater"),
       mu = 0, 
       paired = FALSE, 
       var.equal = FALSE,
       conf.level = 0.95, ...)


Teste-t para uma amosta

t.test(T$Q, mu=10, alternative = "two.sided") 
# H0: mu=10     
# HA: mu!=10


t.test(T$Q, mu=10, alternative = "less") 
# H0: mu=10 
# HA: mu<10


Teste-t para duas amostas emparelhadas

t.test(T$Q,T$Q2,
       paired=TRUE, 
       alternative = "two.sided") 

# H_0: mu1 -mu2 = 0      
# H_A: mu1 -mu2 != 0

\(H_0: mu1 -mu2 = 0\)
\(H_A: \mu1 -\mu2 != 0\)

t.test(T$Q,T$Q2,  paired=TRUE,  alternative = "less")

\(H_A: \mu1 -\mu2 > 0\)

t.test(T$Q,T$Q2,  paired=TRUE,  alternative = "greater")


Teste-t para duas amostas independentes (a)

t.test(T$Q,T$Q2,  alternative="less")             # numérica numérica

Teste t para duas amostas independentes (b)

t.test(Q~C, alternative="less"   , data = T) # numerica e categórica

BETA COEFFICIENTS

[to extract the standardized coefficients from a fitted linear model] (https://stackoverflow.com/questions/24305271/extracting-standardized-coefficients-from-lm-in-r)

A. The easiest way is to just standardize your variables. The coefficients will then automatically be the standardized “beta”-coefficients (i.e. coefficients in terms of standard deviations).

modelo1 <- lm(scale(yQ) ~ scale(xQ), data=T)

B. An alternative: use lm.beta function, from QuantPsyc package

library("QuantPsyc")

modelo2 <- lm( yQ  ~ xQ , data=T)

lm.beta(modelo2)
