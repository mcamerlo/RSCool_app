Criação

  • Standard data.frame
Df <-data.frame (
            id = c(1, 1, 2, 3, 3, 5),
            cigarros = c(15,15, 19, 14, 12, 15),
            sexo = c(0, 0, 1, 1, 0,0),
            cidade = c("Lisboa","Lisboa", "Lisboa", "Porto", "Lisboa", "Portimão"),
            grupo = c("A","A", "C", "C", "B", "B"),

com [ , ] (based on variables’ positions)

Df [1  , ]      # linha 1       , colunas todas

Df [ ,3 ]        # linhas todas, coluna 3

Df [1  ,3 ]         #  linha 1      , coluna 3

Df [1:4, ]  # linhas 1 a 4  , colunas todas

Df [ , 2:5]     # linhas todas  , colunas 2 a 5

Df [c(1,2,3,4), c(2,3,4,5)]

Df [  ,-c(1,2)]  # linhas todas, colunas todas menos 1 e 2

Merge

Integrar duas bases

total <- merge(dataframeA, dataframeB, by="id")
  • all.x
total <- merge(dataframeX, dataframeY, by="id", all.x=TRUE )

Linhas extras serão adicionadas, uma para cada linha em x que não tenha nenhuma linha correspondente em y.

  • all.y
total <- merge(dataframeX, dataframeY, by="id", all.y=TRUE )

Linhas extras serão adicionadas, uma para cada linha em y que não tenha nenhuma linha correspondente em x.

  • all
total <- merge(dataframeA, dataframeB, by="id", all=TRUE )

Integrar linhas

Usar rbind() para integrar dataframes verticalmente

total <- rbind(data frameA, data frameB)

Os bases de dados devem ter as mesmas variáveis, mas não precisam estar na mesma ordem.

Se a base A tiver variáveis que a base B não tem, então:
a) Exclua as variáveis extras na base A; ou.
b). Crie as variáveis adicionais na base B e defina-as como NA antes de integra-las.

Reshape

(mais info aqui)

Os dados são considerados como:
- formato wide (uma linha por sujeito) quando há uma linha de observação por sujeito com cada medição presente como uma variável diferente e
- Formato long (várias linhas por sujeito) quando há uma linha de observação por medição.
